## Examples
### Helloworld
Hello World是一个演示如何使用Solana Javascript API和Rust和C程序来构建，部署，并与Solana区块链上的程序交互的项目。
项目包括：
- 一个链上的hello word 程序
- 一个可以发送 hello 到一个帐户和获取 hello 次数的 客户端

### 编译和运行
获取最新版本的例子代码
```
$ git clone https://github.com/solana-labs/example-helloworld.git
$ cd example-helloworld
```
然后按照[README](https://github.com/solana-labs/example-helloworld/blob/master/README.md)的步骤进行
