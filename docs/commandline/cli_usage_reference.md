## CLI 使用说明
### Examples
#### 获取公钥 / Get Pubkey
```
// Command
$ solana-keygen pubkey

// Return
<PUBKEY>
```
#### 空投SOL/Lamports / Airdrop SOL/Lamports
```
// Command
$ solana airdrop 1

// Return
"1 SOL"
```
#### 获取balance / Get Balance
```
// Command
$ solana balance

// Return
"3.00050001 SOL"
```
#### 确认交易 / Confirm Transaction
```
// Command
$ solana confirm <TX_SIGNATURE>

// Return
"Confirmed" / "Not found" / "Transaction failed with error <ERR>"
```
#### 部署程序/Deploy program
```
// Command
$ solana program deploy <PATH>

// Return
<PROGRAM_ID>
```
